# Atlas Labs HR Analytics with Power BI

## Executive Summary
This project presents a tailored HR analytics solution for Atlas Labs, developed entirely in Power BI. The goal was to provide clear visibility into workforce trends, employee performance, satisfaction levels, and attrition drivers. The report is built on a solid data model and features interactive dashboards powered by validated DAX measures, enabling HR teams and leadership to explore insights and make informed decisions.


## Business Problem
Atlas Labs needed a centralized platform to better understand its workforce and reduce employee turnover. The challenge was to uncover meaningful patterns across demographics, performance, and attrition, and to translate those findings into actionable strategies for improving employee experience and retention.

## Methodology
- Transformed and cleaned data using Power Query
- Designed a Snowflake style data model with a custom DimDate table
- Implemented active/inactive relationships for time-based analysis
- Built modular DAX measures for key performance indicators
- Created four interactive dashboards: Overview, Demographics, Performance Tracker, and Attrition
- Validated all metrics against source data to ensure accuracy and reliability

## Skills and Tools Used
- Power BI for data modeling, dashboard development, and DAX
- Power Query for data preparation
- DAX for business logic and time intelligence
- Microsoft Word for documentation
- Data visualization techniques including cards, charts, slicers, and filters
- Analytical methods for trend analysis, cohort evaluation, and performance tracking

## Results
- Delivered four interactive dashboards tailored to HR priorities
- Enabled filtering by employee, department, and time period
- Highlighted key attrition risks and correlations between satisfaction and performance
- Provided validated KPIs for headcount, attrition, satisfaction, and performance
- Established a scalable framework for ongoing workforce analysis

## Business Recommendations
- Reevaluate policies in departments with elevated attrition rates
- Track overtime and travel frequency as potential stress indicators
- Strengthen manager training and onboarding programs
- Use satisfaction and performance data to shape engagement strategies
- Continue monitoring KPIs to assess the impact of HR initiatives over time
